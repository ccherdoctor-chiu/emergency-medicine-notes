一、核心概念摘要 (Core Concept Summary)

🎯 ATLS三大核心原則

  1. 治療最大的生命威脅優先
  2. 缺乏確診不應延遲緊急治療
  3. 詳細病史非開始治療的必要條件

🔄 主要檢查 (Primary Survey) - xABCDE系統

  • x: eXsanguinating hemorrhage (大出血控制)
  • A: Airway + 頸椎保護 (呼吸道 + Cervical spine protection)
  • B: Breathing (呼吸與通氣)
  • C: Circulation (循環與出血控制)
  • D: Disability (神經功能評估)
  • E: Exposure/Environment (暴露檢查與環境控制)

📋 評估流程架構

初步評估 → 主要檢查 + 立即復甦 → 輔助檢查 → 重新評估 → 次要檢查 → 確定性照護

二、主動回憶問題庫 (Active Recall Question Bank)

Q1: ATLS主要檢查的口訣是什麼？每個字母代表什麼？
A1: xABCDE - eXsanguinating hemorrhage, Airway, Breathing, Circulation, Disability, Exposure
Q2: x(大出血控制)的三大處理方式是什麼？
A2: 直接加壓 (Direct pressure)、傷口填塞 (Wound packing)、止血帶 (Tourniquet)
Q3: 氣道評估時必須同時注意什麼？
A3: 頸椎活動限制 (Cervical spine motion restriction)
Q4: GCS幾分以下通常需要確定性氣道？
A4: GCS ≤ 8分
Q5: 呼吸評估要檢查哪三個組成？
A5: 肺部 (Lungs)、胸壁 (Chest wall)、橫膈 (Diaphragm)
Q6: 休克的定義是什麼？
A6: 組織灌流和氧合不足 (Insufficient tissue perfusion and oxygenation)
Q7: 外傷病人失血的五個解剖位置是什麼？
A7: "One on the floor and four more" - 外出血、胸腔、腹腔、骨盆腔、四肢骨折處
Q8: 低血壓是休克的什麼時期表現？
A8: 相對較晚期的表現，可能已失血達30%
Q9: FAST檢查的適應症是什麼？
A9: 疑似腹部外傷，用於偵測腹膜內出血
Q10: 何時需要重複主要檢查？
A10: 病人狀況任何惡化時都應重複主要檢查

三、記憶輔助工具 (Memory Aids & Mnemonics)

🎭 【xABCDE記憶法】

口訣： "血(x)路(A)不(B)通(C)會(D)餓(E)死"

  • 血：eXsanguinating hemorrhage (大出血)
  • 路：Airway (氣道)
  • 不：Breathing (呼吸)
  • 通：Circulation (循環)
  • 會：Disability (神經功能)
  • 餓：Exposure (暴露檢查)

🩸 【出血控制三步驟】

口訣： "壓包帶"

  • 壓：直接加壓
  • 包：傷口填塞
  • 帶：止血帶

🫀 【失血五處記憶法】

故事聯想： "一個人倒在地板上(外出血)，胸口痛、肚子痛、屁股痛、手腳也在流血"

  • 地板：外出血 (External hemorrhage)
  • 胸：胸腔 (Thoracic cavity)
  • 肚：腹腔 (Peritoneal cavity)
  • 屁股：骨盆和後腹腔 (Pelvic and retroperitoneal space)
  • 手腳：長骨骨折和肌肉間室 (Long bone and muscle compartments)

🧠 【GCS快速判定法】

口訣： "睜眼說話動手腳，4-5-6加起來15"

  • 睜眼：4分
  • 說話：5分
  • 動作：6分
  • 總分：15分

四、概念關聯圖譜 (Concept Connection Map)

🔗 核心邏輯關係

     ↓              ↓              ↓         ↓
優先順序判定    標準化流程      症狀治療    狀況監控
     ↓              ↓              ↓         ↓
資源配置決定    團隊協作      復甦策略    後送決定

📊 評估-治療-監控循環

xABCDE評估 ←→ 立即復甦治療 ←→ 持續重新評估

每個步驟都會影響下一步驟的決策，病人狀況變化時必須回到主要檢查重新開始

💡 考前最後提醒

記住xABCDE不只是檢查順序，更是優先處理的順序 - 發現問題立即處理，不要等到全部檢查完才開始治療！

一、核心進階觀念 (Advanced Core Concepts)

⚠️ 血管加壓劑使用的微妙平衡

關鍵陷阱： 血管加壓劑可用於插管時的血壓支持，但絕不能作為休克的一線治療

  • 正確時機： 快速序列插管 (RSI) 時的暫時性血壓支持
  • 錯誤觀念： 用血管加壓劑來"治療"出血性休克
  • 生理原理： 未控制出血時使用血管加壓劑會加重組織缺血

🔬 ETCO2的雙重意義陷阱

容易忽略： ETCO2反映通氣和灌流兩個面向

  • 低ETCO2 = 可能是早期失血性休克的指標（非僅僅過度換氣）
  • 陷阱情境： 病人ETCO2下降，考生可能只想到呼吸問題，忽略循環問題

🩸 Permissive Hypotension vs TBI的矛盾處置

高階鑑別：

  • 單純穿透性胸腹傷： 允許較低血壓（減少出血）
  • 合併腦外傷： 需要較高血壓目標（維持腦灌流）

年齡分層目標：

  • 小於15歲：SBP ≥ 80 mmHg
  • 15-49歲：SBP ≥ 90 mmHg
  • 50-69歲：SBP ≥ 100 mmHg
  • ≥70歲：SBP ≥ 110 mmHg

二、高階鑑別診斷陷阱 (Advanced Differential Traps)

🫁 張力性氣胸 vs 心包填塞

特徵 張力性氣胸 心包填塞
頸靜脈怒張 有 有
呼吸音 患側消失 雙側正常
叩診 患側鼓音 正常
心音 正常 可能微弱
決定性處置 胸腔減壓 心包穿刺

🩸 外出血控制的層級陷阱

陷阱點： 什麼時候用止血帶？

  • 小軟組織傷 + 表淺動脈出血 → 僅直接加壓
  • 大血管傷 + 多處出血 → 直接上止血帶
  • 加壓/填塞無效的四肢出血 → 應立即上止血帶
  • 頸部/咽部/口腔出血 → 禁用止血帶，只能加壓

三、特殊族群的例外規則 (Special Population Exceptions)

🤱 懷孕患者的隱藏陷阱

  • 生理陷阱： 可承受更大血容量流失才出現低血壓
  • 解剖陷阱： 子宮壓迫下腔靜脈 → 左側傾斜是關鍵處置
  • 監測陷阱： 胎兒窘迫可能比母體症狀更早出現

👴 老年患者的"正常"血壓陷阱

關鍵觀念： 有高血壓病史的老年人，"正常"血壓可能就是低血壓

實例： 平常血壓160/90的病人，血壓120/80可能已經是相對低血壓

處置差異： 需要更高的血壓目標來維持器官灌流

🏃‍♂️ 運動員的生理儲備掩蔽效應

  • 陷阱： 不會出現早期休克症狀（心跳過快、呼吸急促）
  • 靜息血壓： 可能比一般人低
  • 危險性： 容易低估傷勢嚴重程度

👶 小兒患者的急劇惡化陷阱

  • 生理特性： 強大代償能力 + 急劇失代償
  • 體溫調節： 散熱快，容易低體溫 → 影響凝血功能
  • 劑量陷阱： 所有藥物/輸液都需要依體重計算

四、程序性陷阱與禁忌症 (Procedural Traps & Contraindications)

🚫 導尿管置放的絕對禁忌

禁忌症識別：

  • 尿道口出血
  • 會陰部瘀血
  • 前列腺上移觸診

陷阱： 在有禁忌症時強行插管會造成尿道完全撕裂

🚫 鼻胃管插管的隱藏禁忌

  • 禁忌症： 已知或疑似顱底骨折（篩骨骨折）
  • 風險： 鼻胃管可能插入腦內
  • 替代方案： 改用口腔插管

五、主動回憶陷阱題庫 (Advanced Trap Questions)

Q1: 病人插管時可以使用血管加壓劑嗎？這與休克治療時使用有何不同？
A1: 可以用於插管時暫時血壓支持，但絕不能用來治療出血性休克本身
Q2: ETCO2突然下降，除了換氣問題還要考慮什麼？
A2: 循環問題 - 可能是早期失血性休克的表現
Q3: 腦外傷合併胸部穿透傷的血壓目標如何制定？
A3: 以腦外傷的較高血壓需求為準，不能採用permissive hypotension
Q4: 什麼情況下強行置放導尿管是危險的？
A4: 尿道口出血、會陰瘀血時，可能造成尿道完全撕裂
Q5: 為什麼運動員的外傷評估特別需要小心？
A5: 優異體能可能掩蓋早期休克症狀，容易低估傷勢
Q6: 懷孕患者出現低血壓時的第一個處置是什麼？
A6: 左側傾斜，解除子宮對下腔靜脈的壓迫

🎯 考前陷阱提醒

  1. 不要被"正常"數值騙了 - 特殊族群有不同基準
  2. 處置衝突時優先考慮 - 哪個威脅更immediate？
  3. 新舊觀念並存時 - 以最新evidence-based指引為準
  4. 監測工具都有盲點 - 臨床判斷永遠是王道

一、緊急狀況快速應變 (Rapid Emergency Response)

🚨 張力性氣胸 (Tension Pneumothorax)

核心判斷點: 呼吸窘迫 + 患側呼吸音消失 + 頸靜脈怒張 + 氣管偏移

立即SOP:

  1. 第5肋間鎖骨中線 或 第2肋間鎖骨中線 立即針刺減壓
  2. 使用 14G針頭 (粗針)
  3. 聽到氣體洩出聲 = 成功
  4. 立即準備胸管置放

🩸 大量外出血控制

核心判斷點: 動脈性出血 + 傳統止血法無效

立即SOP:

  1. 四肢出血: 立即上止血帶於出血點近心端
  2. 頸部/軀幹出血: 直接加壓 + 紗布填塞
  3. 頭皮出血: 環形縫合或血管夾快速止血
  4. 止血帶位置: 距離傷口5-7公分，記錄上帶時間

💨 呼吸道完全阻塞

核心判斷點: 無法發聲 + 呼吸費力 + 發紺

立即SOP:

  1. 清醒病人: 哈姆立克法 (Heimlich maneuver)
  2. 昏迷病人: 下顎推舉 (Jaw thrust) + 異物移除
  3. 無效時: 準備緊急環甲膜切開術
  4. 頸椎受傷: 僅用下顎推舉，禁用頭部後仰

🫀 心跳停止 (Cardiac Arrest)

核心判斷點: 無脈搏 + 無呼吸 + 無反應

立即SOP:

  1. 立即CPR: 胸骨下1/3，深度5-6公分，頻率100-120/分
  2. 可電擊心律: 立即除顫
  3. 外傷性心跳停止: 考慮張力性氣胸、心包填塞、大量失血
  4. ETCO2監測: <10mmHg提示CPR品質不佳

二、臨床常用劑量與給藥速查 (Common Dosing & Administration Quick Ref.)

💉 Epinephrine (腎上腺素)

  • 過敏性休克: 0.3-0.5mg IM (大腿外側) / 可重複每5-15分鐘
  • 心跳停止: 1mg IV/IO 每3-5分鐘
  • 特別提示: IM注射比皮下注射吸收更快

🩸 Tranexamic Acid (傳明酸)

  • 外傷性出血: 1g IV over 10分鐘，之後1g IV over 8小時
  • 給藥時機: 受傷後3小時內最有效
  • 特別提示: 超過3小時給藥可能增加死亡率

💨 Oxygen (氧氣)

  • 初始給氧: 所有外傷病人都給予100%氧氣
  • 目標SpO2: ≥94% (COPD病人≥88%)
  • 給氧裝置: Bag-mask reservoir 可達到90-95%氧氣濃度
  • 特別提示: 不要因為怕CO2滯留而限制氧氣

🧠 Mannitol (甘露醇)

  • 顱內壓升高: 0.25-1g/kg IV push
  • 給藥速度: 快速推注 (5-10分鐘內)
  • 特別提示: 需要確認腎功能正常，避免腎衰竭

💔 Norepinephrine (正腎上腺素)

  • 插管時血壓支持: 0.1-0.5 mcg/kg/min IV
  • 神經性休克: 初始劑量0.5-1 mcg/min
  • 特別提示: 絕不用於出血性休克的主要治療

三、關鍵評估與判讀要點 (Key Assessments & Interpretation)

🧠 GCS快速評估法

評估目的: 意識程度量化 + 預後判斷

快速判讀:

  • 輕度: 13-15分 (可能腦震盪)
  • 中度: 9-12分 (需要密切監測)
  • 重度: 3-8分 (需要插管保護氣道)

代表意義: ≤8分通常需要確定性氣道

📊 Shock Index (休克指數)

評估目的: 早期休克偵測

計算公式: 心跳數 ÷ 收縮壓

快速判讀:

  • 正常: 0.5-0.7
  • 輕度休克: 0.7-1.0
  • 重度休克: >1.0

代表意義: >1.0需要積極輸液復甦

🔍 FAST檢查標準位置

評估目的: 腹腔內出血快速偵測

四個標準位置:

  1. 心包膜: 劍突下切面
  2. 右上腹: 肝腎間隙 (Morrison's pouch)
  3. 左上腹: 脾腎間隙
  4. 骨盆腔: 膀胱直腸間隙 (男) / 膀胱子宮間隙 (女)

判讀要點: 任一處有液體堆積 = 陽性

📈 End-tidal CO2 (ETCO2)

評估目的: 通氣 + 循環狀態監測

正常值: 35-45 mmHg

快速判讀:

  • <25 mmHg: 可能過度換氣或循環衰竭
  • >50 mmHg: 可能換氣不足
  • 突然下降: 考慮肺栓塞、心跳停止、插管滑脫

CPR監測: >10 mmHg提示有效胸壓

四、實務操作核心提示 (Practical Procedure Pointers)

🫁 緊急插管準備 (Emergency Intubation)

SOAP-ME準備:

  • Suction (抽吸設備)
  • Oxygen (預給氧)
  • Airway equipment (氣道設備)
  • Pharmacy (藥物)
  • Monitoring (監測)
  • Equipment backup (備用設備)

關鍵步驟:

  1. 頸椎保護: 助手維持頸椎中立位
  2. 預給氧: 3-5分鐘100%氧氣
  3. 備用計畫: 準備LMA或緊急氣道

安全提示: 外傷病人都當作困難氣道處理

🩸 中心靜脈導管置放

關鍵步驟:

  1. 位置選擇: 鎖骨下 > 頸內 > 股靜脈
  2. 超音波導引: 即時監看血管結構
  3. 確認位置: 胸部X光確認導管位置
  4. 併發症監視: 氣胸、動脈穿刺、感染

效率訣竅: 先用小針頭確認位置再用大針

🚰 導尿管置放技巧

關鍵步驟:

  1. 禁忌症確認: 無尿道口出血、會陰瘀血
  2. 潤滑劑: 充分使用含局部麻醉劑潤滑劑
  3. 男性技巧: 陰莖向上拉直，克服尿道彎曲
  4. 阻力處理: 遇阻力不可強行推進

安全提示: 疑似尿道損傷時絕對禁止插管

🍽️ 鼻胃管插管

關鍵步驟:

  1. 禁忌症: 顱底骨折 (改用口腔插管)
  2. 插管深度: 鼻翼到耳垂到劍突的距離
  3. 位置確認: 抽吸胃液 + 胸部X光確認
  4. 固定方式: 避免壓迫鼻翼造成潰瘍

效率訣竅: 插管時請病人做吞嚥動作

🦴 脊椎固定要點

關鍵步驟:

  1. 頸椎固定: C-collar + 人工維持中立位
  2. 翻身技巧: 至少4人log roll
  3. 脊椎板限制: 僅用於轉運，不可長時間留置
  4. 壓瘡預防: 每2小時檢查壓力點

安全提示: 脊椎板本身不提供脊椎固定功能

五、交班重點速記 (Handoff Communication)

📞 S-xABCDE-BAR交班法

S (Situation): 轉診醫師+醫院、病人基本資料、受傷機轉、轉診適應症

xABCDE:

  • x: 止血帶、加壓、填塞等出血控制
  • A: 氣道評估、插管狀況
  • B: 呼吸評估、胸管、氧氣需求
  • C: 循環評估、輸液、輸血狀況
  • D: 神經評估、GCS、瞳孔反應
  • E: 體溫、其他發現

B (Background): AMPLE病史、已給藥物、影像檢查

A (Assessment): 目前生理狀況、治療反應、可能診斷

R (Recommendation): 運送方式、運送中可能需要的處置

六、常見併發症立即警示 (Immediate Complication Alerts)

⚠️ 輸血反應

立即警示徵象: 發燒、發冷、背痛、血尿

應變SOP:

  1. 立即停止輸血
  2. 保持靜脈通路暢通 (生理食鹽水)
  3. 檢查血袋標籤與病人身分
  4. 通知血庫與主治醫師
  5. 留取血液和尿液檢體

💧 點滴外滲

立即警示徵象: 注射部位腫脹、疼痛、皮膚變色

應變SOP:

  1. 立即停止輸液
  2. 拔除靜脈導管
  3. 抬高患肢
  4. 冰敷15-20分鐘 (血管收縮劑外滲用溫敷)

🔌 導管阻塞

中心靜脈導管阻塞:

  1. 嘗試抽吸 (不可強力沖洗)
  2. 使用溶栓劑 (如許可)
  3. 考慮更換導管

導尿管阻塞:

  1. 檢查導管是否彎曲
  2. 膀胱沖洗 (生理食鹽水)
  3. 必要時更換導管

七、快速決策流程圖 (Quick Decision Trees)

🩸 出血控制決策樹

                        外出血 
                        ├── 四肢 → 加壓無效 → 止血帶
                        ├──
                        軀幹 → 加壓 + 填塞 → 手術
                        └── 頭頸 → 加壓 + 縫合 → 血管夾

🫁 呼吸窘迫決策樹

                    呼吸窘迫
├── 單側呼吸音消失 → 氣胸 → 胸腔減壓
├── 雙側囉音 → 肺水腫 → 利尿劑
└── 呼吸音正常 → 上呼吸道阻塞 → 氣道處理

💔 休克決策樹

                    休克
├── 出血性 → 止血 + 輸血
├── 心因性 → 強心劑 + 利尿
├── 感染性 → 抗生素 + 輸液
└── 神經性 → 血管加壓劑

  2. 安全第一 - 自己安全才能救人
  3. 標準化流程 - 壓力下靠SOP不靠記憶
  4. 持續監測 - 病人狀況隨時在變化
  5. 團隊溝通 - 大聲、清楚、確認

二、圖表二：Figure 1-2: National Guideline for the Field Triage

【RED CRITERIA (高風險嚴重傷害)】

傷害型態 意識狀態與生命徵象
• 穿透性傷害：頭、頸、軀幹、近端肢體
• 顱骨變形、疑似顱骨骨折
• 疑似脊髓傷合併新發運動或感覺喪失
• 胸壁不穩定、變形、疑似連枷胸
• 疑似骨盆骨折
• 疑似兩個以上近端長骨骨折
• 肢體壓碎、撕脫、變形或無脈搏
• 腕關節或踝關節以上截肢
• 需要止血帶或傷口填塞持續加壓的活動性出血		 全部病人：
• 無法遵從指令 (運動性GCS <6)
• RR <10 或 >29 次/分
• 呼吸窘迫或需要呼吸支持
• 室內空氣脈搏血氧飽和度 <90%

年齡分層血壓標準：
• 0-9歲：SBP < 70 + (2 × 年齡) mmHg
• 10-64歲：SBP < 90 mmHg 或 HR > SBP

【YELLOW CRITERIA (中度風險嚴重傷害)】

受傷機轉 EMS判斷
高風險車禍：
• 部分或完全拋出車外
• 明顯車體變形
• 乘客車廂內死亡
• 兒童未固定或安全座椅未固定

其他機轉：
• 騎乘者與運輸工具分離
• 行人/騎士被拋擲、輾過
• 高度墜落 >10英尺		 考慮風險因子：
• 幼兒/老年人低位墜落
• 抗凝血劑使用
• 疑似兒童虐待
• 特殊高資源醫療需求
• 懷孕 >20週
• 燒燙傷合併外傷
• 兒童優先轉至小兒專科中心

【圖表核心解釋】

此圖建立二階段篩檢系統：紅色標準決定「必須送最高等級外傷中心」，黃色標準決定「優先送外傷中心」。核心邏輯是生理異常 > 解剖傷害 > 機轉判斷 > EMS經驗的優先順序。

【臨床應用與記憶建議】

臨床應用： 院前人員或急診分檢時的決策樹，按順序檢查：先看生命徵象，再看明顯傷害，最後考慮機轉和特殊情況。

記憶策略： 用紅綠燈概念：紅燈=立即最高級，黃燈=優先但可調整。記住數字「6-8-90」：GCS運動<6、年齡0-9歲血壓公式、10歲以上血壓<90。
